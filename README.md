# main-portal-notes

本项目用于归档和发布多个独立的 HTML 页面。这些页面通常作为临时笔记、工具页面、信息展示页或辅助入口使用，不针对任何特定网站或服务。

## 项目简介

`main-portal-notes` 是一个静态页面集合仓库，所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。方便快速部署到静态托管平台，用于分享或存档某些内容。

## 目录说明

```
.
├── README.md           # 本文件
├── index.html          # 仓库入口页面（可选）
├── pages/              # 页面归档目录
│   ├── example.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── docs/               # 相关文档或说明（可选）
```

- `index.html`：可选的主入口页面，可按需创建。
- `pages/`：存放所有归档的 HTML 页面文件。
- `assets/`：页面所需的样式、脚本及图片资源。
- `docs/`：补充说明文档。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接通过浏览器打开。
- 页面之间可能相互引用，但建议保持相对路径结构。
- 页面内容以信息展示、工具或笔记为主，不包含商业或推广性质的内容。
- 如需新增页面，请在 `pages/` 目录下创建，并确保资源路径正确。
